#   - aircrack-ng : suite untuk auditing Wi-Fi (mengandung airodump-ng, aireplay-ng)

#   - sudo        : (opsional tapi direkomendasikan) untuk menjalankan dengan hak root
#
# Contoh instalasi (Debian / Kali Linux):
#   sudo apt update
#   sudo apt install -y aircrack-ng xterm sudo
#
# Contoh instalasi (Termux):
#   pkg update
#   pkg upgrade
#   pkg install -y git python aircrack-ng xterm termux-tools
#
# Catatan penting:
# - Pastikan adapter Wi-Fi Anda mendukung mode monitor (monitor mode).
# - Jalankan script dengan hak akses root: `sudo bash Ghost-Vuln-Scanner-Wifi.sh`
# - Script akan membuat file temp (ghost_v19_intel-01.csv) dan laporan akhir
#   (GHOST_FINAL_AUDIT.txt).
